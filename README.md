# Python Learning Repository

This repository contains basic Python coding examples and exercises for learning fundamental programming concepts.

## Project Structure

```
.
├── 01_basics/
│   ├── hello_world.py
│   ├── variables_and_types.py
│   ├── operators.py
│   └── user_input.py
├── 02_control_flow/
│   ├── if_else.py
│   ├── loops.py
│   └── break_continue.py
├── 03_functions/
│   ├── basic_functions.py
│   ├── parameters_and_return.py
│   └── scope.py
├── 04_data_structures/
│   ├── lists.py
│   ├── tuples.py
│   ├── dictionaries.py
│   └── sets.py
├── README.md
└── requirements.txt
```

## Getting Started

1. Clone this repository
2. Navigate to the project directory
3. Run Python files to see examples

```bash
python 01_basics/hello_world.py
```

## Topics Covered

- Variables and Data Types
- Operators (Arithmetic, Logical, Comparison)
- Control Flow (if/else, loops)
- Functions and Scope
- Data Structures (Lists, Tuples, Dictionaries, Sets)
- String Manipulation
- File Handling
- Exception Handling
- Object-Oriented Programming (Classes and Objects)

## Requirements

- Python 3.8 or higher

## Usage

Each file is a standalone example that can be run independently:

```bash
python filename.py
```
